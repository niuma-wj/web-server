# 项目简介

本项目为 NiuMa 游戏平台 WEB 后端服务程序，实现了客户端登录以及主界面的业务功能，同时还实现了游戏管理后台的相关业务逻辑功能。\
本项目基于若依开源框架开发，若依是基于 SpringBoot+Vue 前后端分离的 Java 快速开发框架，感谢若依开源项目开发者的无私奉献。若依项目地址：[RuoYi](https://gitee.com/y_project/RuoYi)
